Sistema de Gestión de Créditos

Registra, administra y visualiza créditos de forma sencilla y eficaz.

¿Para qué sirve este programa?

Este sistema fue creado con el objetivo de facilitar el registro, control y visualización de créditos otorgados a clientes. Permite almacenar información como nombre del cliente, monto del crédito, tasa de interés, plazo y fecha de otorgamiento.

Tecnologías utilizadas

  • Python con Flask para el backend
  • SQLite como base de datos ligera
  • HTML5 y Bootstrap Lux para la interfaz
